# Dear.Green

## 🌏 서비스 기획서: Dear Earth (친애하는 지구에게)
"완벽한 한 명보다, 불완전한 다수의 다정한 발걸음이 지구를 숨 쉬게 합니다."

## 프로젝트 개요
서비스명: Dear Earth (친애하는 지구에게)

슬로건: 당신의 식탁과 지구 사이, 다정한 거리를 만드는 기록

핵심 철학: 완벽함보다는 **불완전한 실천** 의 가치를 존중합니다.

먹이사슬을 부정하기보다 생명과의 **적정한 거리**를 인지합니다.

죄책감이 아닌 **재미와 연대**를 통해 지속 가능한 채식 지향 삶을 돕습니다.

## 🛠️ Dear Earth MVP 핵심 스펙 (Vercel 배포용)
Vercel은 Next.js나 React 기반의 정적/서버리스 사이트에 최적화, 아래 3가지 기능 사용

### 한 끼 로그 (The Log)
복잡한 칼로리 계산은 빼고, **나의 선택**에만 집중합니다.

기능: 아침/점심/저녁 버튼 클릭 시 팝업 노출.

단계 선택: [비건 / 오보-락토 / 페스코 / 폴로 / 플렉시테리언] 중 선택.

타협 기록: 만약 고기를 먹었다면 "그래도 채소를 많이 먹었나요?" 같은 체크박스 1~2개 추가.

결과: 선택에 따라 '인지 포인트(IP)' 지급.

### 시각적 성장 (The Forest & Blob)
사용자가 가장 큰 재미를 느낄 부분입니다.

캐릭터: 중앙에 '지구 덩어리(Blob)' 캐릭터 배치.

성장 로직: IP가 일정 점수 쌓일 때마다 캐릭터 주변에 풀이 돋거나, 나무가 한 단계 커짐 (이미지 3~5단계 교체 방식).

인터랙션: Framer Motion 같은 라이브러리를 써서 캐릭터가 둥둥 떠다니는 효과만 주어도 '지브리 감성'이 살아납니다.

### 10초 숨 고르기 (Mindfulness Timer)
사이트의 철학을 보여주는 가장 쉬운 방법입니다.

기능: 식사 기록 버튼을 누르기 전, "잠시만요" 팝업과 함께 10초 카운트다운.

내용: "이 음식이 어디서 왔는지 10초만 생각해보세요." 라는 문구 노출.

보상: 타이머를 끝까지 보면 추가 포인트 지급.


## 💻 추천 기술 스택 (Vercel 최적화)
| **구분** | **추천 도구** | **이유** |
| --- | --- | --- |
| **Framework** | **Next.js** | Vercel 제작사에서 만든 프레임워크라 배포가 가장 쉽습니다. |
| **Styling** | **Tailwind CSS** | 따뜻한 어스 톤(Earth Tone)을 빠르게 입히기에 좋습니다. |
| **Database** | **Supabase** | 구글 로그인 연동과 데이터 저장이 매우 빠르고 무료 티어가 넉넉합니다. |
| **Animation** | **Framer Motion** | 캐릭터의 움직임과 숲의 성장을 부드럽게 표현해 줍니다. |

## 📅 개발 우선순위 (Roadmap)
1주차: 기본 레이아웃 구성 + 식단 선택 버튼 구현.

2주차: Supabase 연동 (로그인 및 기록 저장).

3주차: 캐릭터 성장 로직(이미지 교체) + 10초 타이머 추가.

4주차: 디자인 디테일 수정 및 Vercel 배포 (클릭 한 번이면 끝!).
  
## 기획자 노트 (Planner's Note)
"Dear Earth는 사용자에게 '비건이 되라'고 강요하지 않습니다. 대신 식탁 앞에서 잠시 멈춰 서서 **'이 생명은 어디서 왔을까?'**라고 묻게 만듭니다. 그 10초의 고민이 모여 숲을 이루고, 그 숲이 곧 우리의 지구가 됩니다. 우리는 가장 즐거운 방식으로 그 여정에 동행할 것입니다."
